The Network

  • OpenVPN Configuration Files: cryptostorm.org/conf or github.com/…/conf
  • Windows Widget: cryptostorm.org/widget
  • Cryptofree: cryptofree.me
    Free access to the Cryptostorm Private Network (same features, limited speed).
  • Token Hasher: cryptostorm.is/#section6
    Tokens have to be SHA512-hashed before being usable in OpenVPN or the Windows widget.
  • Token Checker: cryptostorm.nu
    Check validity and duration of tokens.
  • VPN Nodes and DNS Server IPs: github.com/…/NodesOfCryptostorm.txt
  • VPN Connection IP Checker: cryptostorm.is/test.cgi
    Check if you're connected to the Cryptostorm Darknet.
  • How-To's and Tutorials: cryptostorm.org/howto
    OpenVPN setup tutorials for various operating systems and other how-to's.
  • Goodies: cryptostorm.is/goodies
    A collection of tools, projects, and haberdasheries.
  • Torstorm: torstorm.org
    Browser-based access to .onion sites. Simply replace the ".onion" part of an URL with ".torstorm.org".
  • STUNnion: stunmbj4vvnuv5pr.onion
    WebRTC IP leak test for the Tor network.
